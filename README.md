# Faceless Marketplace

A trusted marketplace MVP for South African buyers and everyday sellers.

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS
- Supabase Auth, Postgres, Storage, and RLS
- Vercel
- Cloudflare DNS

## Development

```bash
npm install
npm run dev
```

Before committing:

```bash
npm run lint
npm run build
```
